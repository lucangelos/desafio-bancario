# 💰 Desafio Bancário em Java

Este projeto foi desenvolvido como parte do meu primeiro curso de Java na Alura, com o objetivo de reforçar conceitos fundamentais da linguagem, como:

- Tipos primitivos
- Variáveis
- Estruturas condicionais (`switch case`)
- Estruturas de repetição (`while`)
- Entrada de dados com `Scanner`
- Operações matemáticas básicas

## 📖 Sobre o Projeto

O sistema simula uma conta bancária simples através do terminal. Ao iniciar a aplicação, são exibidos os dados de um cliente, incluindo:

- Nome
- Tipo de conta
- Saldo inicial

Em seguida, o usuário pode interagir com um menu de operações bancárias.

## ⚙️ Funcionalidades

### 1️⃣ Consultar saldo
Exibe o saldo atual da conta.

### 2️⃣ Receber valor
Permite adicionar um valor ao saldo da conta.

### 3️⃣ Transferir valor
Permite transferir um valor, descontando-o do saldo disponível.

### 4️⃣ Sair
Encerra a aplicação.

## 🖥️ Exemplo de Execução

```text
DADOS PESSOAIS

Nome: Lucas Ângelo
Tipo de Conta: Corrente
Saldo Bancário: 250.0
***********************************

Operações

1- Consultar saldos
2- Receber valor
3- Transferir valor
4- Sair

```

## 🚀 Como Executar o Projeto

### 📋 Pré-requisitos

Certifique-se de ter o Java JDK instalado em sua máquina.

Verifique a instalação executando:

```bash
java --version
javac --version

```
### 📥 Clonando o Repositório

git clone https://github.com/lucangelos/desafio-bancario.git
cd desafio-bancario

### ⚙️ Compilando e Executando

javac Main.java
java Main
